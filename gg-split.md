# gg split

Split a single large commit into multiple smaller, logical commits.

---

## What it does

`gg split` helps you break down a big commit into clean, meaningful commits.  
This improves commit history, makes code reviews easier, and keeps pull requests organized.

---

## When to use it

- When a commit contains unrelated or mixed changes  
- Before opening a pull request  
- To improve commit readability and maintainability  
- When reviewers request cleaner commit history  

---

## Usage

```bash
gg split
